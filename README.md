Über dieses Projekt
Dieses Projekt ist mein praktisches Portfolio als Vibe Coder. Ich baue nicht „perfekt auf dem Papier“, sondern arbeite Schritt für Schritt: Idee, Umsetzung, Test, Verbesserung, Deployment.

Worum es hier geht
Die Website zeigt, wie ich moderne Seiten mit KI-Unterstützung aufbaue und produktiv online bringe. Der Fokus liegt auf klarer Wirkung, guter Lesbarkeit und einem sauberen Live-Stand.

Was ich konkret gemacht habe
Eine getrennte neue Projektversion aufgebaut, damit die alte Website sauber unangetastet bleibt.
Erste Animationen auf Anime.js umgestellt.
Den Kontaktbereich vereinfacht (direkte Kontaktinfos statt Formular).
Inhalte verständlicher und ehrlicher formuliert.
Eine eigene, ausführlichere Case-Sektion ergänzt.
Das Design einzelner Bereiche überarbeitet, damit es moderner und klarer wirkt.
Build erstellt und erfolgreich auf Cloudflare Pages veröffentlicht.
Danach den Stand auf GitHub Main gebracht.
Wie ich arbeite
Ich arbeite in kleinen, nachvollziehbaren Schritten.
Ich prüfe Änderungen direkt lokal im Browser.
Ich verbessere iterativ statt alles in einem großen Wurf.
Ich dokumentiere den Verlauf, damit Entscheidungen nachvollziehbar bleiben.
