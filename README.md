## Über dieses Projekt

Dieses Projekt ist mein praktisches Portfolio als Modern Developer.
Ich baue nicht „perfekt auf dem Papier“, sondern arbeite Schritt für Schritt: Idee, Umsetzung, Test, Verbesserung, Deployment.

### Worum es hier geht

Die Website zeigt, wie ich moderne Seiten mit modernen Workflows aufbaue und produktiv online bringe.
Der Fokus liegt auf klarer Wirkung, guter Lesbarkeit und einem sauberen Live-Stand.

### Was ich konkret gemacht habe

1. Eine getrennte neue Projektversion aufgebaut, damit die alte Website sauber unangetastet bleibt.
2. Erste Animationen auf Anime.js umgestellt.
3. Den Kontaktbereich vereinfacht (direkte Kontaktinfos statt Formular).
4. Inhalte verständlicher und ehrlicher formuliert.
5. Eine eigene, ausführlichere Case-Sektion ergänzt.
6. Das Design einzelner Bereiche überarbeitet, damit es moderner und klarer wirkt.
7. Build erstellt und erfolgreich auf Cloudflare Pages veröffentlicht.
8. Danach den Stand auf GitHub Main gebracht.

### Wie ich arbeite

- Ich arbeite in kleinen, nachvollziehbaren Schritten.
- Ich prüfe Änderungen direkt lokal im Browser.
- Ich verbessere iterativ statt alles in einem großen Wurf.
- Ich dokumentiere den Verlauf, damit Entscheidungen nachvollziehbar bleiben.

### Eingesetzte Tools

### Aktueller Stand

- Lokal lauffähig
- Produktions-Build erfolgreich
- Auf Cloudflare veröffentlicht
- Auf GitHub Main aktualisiert

### Kurz gesagt

Ich bin kein klassischer „Senior Engineer“, sondern ein pragmatischer Modern Developer:
Ich lerne schnell, setze zuverlässig um und liefere funktionierende Ergebnisse, die online sichtbar sind.
