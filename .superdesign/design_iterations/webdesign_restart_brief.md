# Webdesign Restart Brief

Ziel: Die Unterseite `/webdesign` wird gestalterisch komplett neu gedacht.

Fokus:
- Webdesign-Dienstleistungen fuer kleine Unternehmen und Selbststaendige
- SEO, Performance, einfache Ablaeufe, klare Kontaktfuehrung
- modern, hochwertig, serioes
- keine generische AI-Landingpage
- kein Dashboard-Grid, keine SaaS-Kachelwand

Form:
- nur eine Screen-Komposition pro Variante
- responsive fuer mobile, tablet, desktop
- geeignet als Ausgangspunkt fuer spaetere React-Umsetzung in Vite + TypeScript
- Superdesign-Canvas-kompatibel: HTML in `.superdesign/design_iterations`

Gewuenschte Richtung:
- editorial, posterartig, mit starkem Hero
- algorithmic-art oder grafische Signal-/Raumlogik statt Stock-Feeling
- klare Typohierarchie
- scanbare Sektionen
- direkte Anfragefuehrung
