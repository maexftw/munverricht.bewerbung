# ASCII Glitch Ripple Hover Effect

A Pen created on CodePen.

Original URL: [https://codepen.io/erevan/pen/MYKBjdZ](https://codepen.io/erevan/pen/MYKBjdZ).

ASCII text glitch ripple on hover for links created for my sub-10kb portfolio website.. Vanilla JS, minimal CSS, no dependencies. 
The effect launches waves from the cursor, scrambles characters using a configurable ASCII map. 
Options include duration, spread, and space preservation. Works on long labels.
